## Brainstorm Sessions

### Skill Resource Economy — 2026-05-11
- **Surface question:** What replaces mana points for using Skills?
- **Better question surfaced:** How do resources make combat rhythm feel like it's building toward something, rather than just filling a meter until a button unlocks?
- **Prior art consulted:** FFXIV Samurai (Sen system), MTG (colored mana identity), Path of Exile (Power/Frenzy/Endurance charges), Hades (cast bloodstone chaining)
- **Directions on the table:**
  - Option A — Direct Stance-Lock: stance actions generate that stance's color tokens; skills spend color combos
  - Option B — Switch Bonus: same as A but switching stances also generates 1 token of the stance you left
  - Option C — Resonance Pairs: resources have meaning as combinations; resonance is a condition (not a currency) gating skills
- **Decided / leaning:** Option C — Resonance Pairs, with Fallacy and Paradox as the two skill-generated resources
- **Open questions:**
  - What threshold triggers resonance? *Implicit — resonance is just holding tokens of two types simultaneously; the skill's cost object itself sets the threshold.*
  - Are Fallacy/Paradox tokens spent on use or do they persist as a state? *Spent on use (deducted from `combatResources.fallacy` / `.paradox`).*

### Locked design

**5 resources:**

| Resource | Symbol | Generated by |
|---|---|---|
| Heart | ♥ | Heart stance basic actions |
| Body | ⚡ | Body stance basic actions |
| Mind | ★ | Mind stance basic actions |
| Fallacy | ⚖ | Using a Fallacy-category skill |
| Paradox | ∞ | Using a Paradox-category skill |

**Resonance** is a *condition*, not a currency. You are in Body-Heart Resonance when you currently hold tokens of both types. Skills require a resonance state to unlock, then spend specific token amounts. Spending the tokens dissolves the resonance.

**Skill cost tiers:**
- Tier 1: Single stance color (e.g. `{ body: 3 }`). Fires early. Generates ⚖ or ∞ on use.
- Tier 2: Resonance-gated (e.g. `{ body: 2, heart: 2 }`). Requires mixed-stance play.
- Tier 3: Philosophical cost (e.g. `{ mind: 2, fallacy: 1 }`). Requires a prior Tier 1 skill this combat.

**Implementation note:** `Skill.manaCost: number` → `Skill.resourceCost: ResourceCost` where
`ResourceCost = { heart?: number; body?: number; mind?: number; fallacy?: number; paradox?: number }`.
`Character.mana` / `Character.maxMana` removed. Anyone working on `specs/04-skills-engine.md`
should not design around `manaCost`.

## Difficulty System
- No upfront difficulty selection.
- Difficulty is a meter driven by moral choices:
  - Moral choices increase it; immoral choices decrease it.
- Example: Escort a child from start to end.
  - Opens one possible ending, but closes another.
  - Having someone murder the child: the only path to the happy ending is to forgive the murderer.

## Combat — Tier 2 Synergy
- Tier 1 effects reward repeating the same Stance. Tier 2 should reward switching.
- Add skills that get bonuses based on the **duration or intensity** of an effect already on the field.

**Example A — Duration cross-type:**
Enemy has 5 rounds of poison. Player uses a different Stance skill that doubles poison duration.
If the post-doubled duration is ≥ 5, deal bonus damage equal to the new duration.

**Example B — Buff type-swap:**
Player has 5+ rounds of Thorns (Body). Uses a Heart skill to convert Thorns to a heart-typed effect.
If duration ≥ 5, Thorns become a Bat Swarm: instead of dealing reflected damage, it heals the player based on the remaining duration.
