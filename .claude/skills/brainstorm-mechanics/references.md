# Prior-Art Reference Catalog

Curated list of games and the *specific* mechanics that map onto problem
shapes Axiomancer Mechanics keeps running into. Cite **game name + named
mechanic**, never vague references. Player reception notes describe
aggregate community discourse (forums, postmortems, designer talks) — if
you don't have a clean read for a given mechanic, say so instead of
inventing one.

---

## 1. RPS / type-triangle combat

Axiomancer's Heart > Body > Mind > Heart loop.

| Game | Mechanic | What worked | What didn't |
|---|---|---|---|
| Pokémon | 18-type matchup chart | Type advantage as identity puzzle; team building becomes meta-game. | Once a player owns coverage for all types, RPS dissolves into "always have the right tool". |
| Fire Emblem | Weapon triangle (sword > axe > lance > sword) | Made unit positioning a tactical question. | Series eventually removed it (Three Houses) because it punished diverse armies. Players were split: some missed it, some celebrated. |
| Triangle Strategy | Three-faction conviction triangle | RPS extended to *narrative/political* axes, not just combat. | Some players felt votes were railroaded by hidden scoring. |
| Yu-Gi-Oh (early) | Attribute Fire/Water/Wind in TCG | Mostly flavor; rarely mechanically loadbearing. | Players treat it as set-dressing. |
| MTG | Five-color wheel (allied/enemy pairs) | Color identity carries theme + mechanical permission. Loved. | Mono-color decks can feel pigeonholed; "good stuff" piles dilute identity. |

**Design takeaway for Axiomancer:** The triangle is sturdiest when each
side has *distinct verbs*, not just +2/−2 on the same verb. Sekiro and
Fire Emblem fans both bounce off triangles that are pure damage modifiers.

## 2. Stance / posture switching

Tier 2 reward-for-switching problem.

| Game | Mechanic | Notes |
|---|---|---|
| Sekiro: Shadows Die Twice | Posture meter; deflect vs dodge stances | Switching *changes what inputs do*. Praised as transformative; "deflect Tuesday" memes. |
| Nioh / Nioh 2 | High/Mid/Low stance triangle on the same weapon | Each stance is a different moveset, not a stat tweak. Praised for depth, criticized for input complexity. |
| Final Fantasy XIV — Samurai | Sen gauge (3 named sen → Iaijutsu) | Switching ↔ committing has a clear payoff curve. Loved for "rotation feel". |
| Devil May Cry 5 | Style ranks rise with variety, drop with repetition | Mechanically rewards switching attacks every few seconds; gives weight to *not* repeating. |
| Magic: the Gathering | Cumulative upkeep / "must change land each turn" lands | Forces variety via cost escalation. |

**Design takeaway:** "Switching is rewarded" works best when the stances
have *different verbs* (Sekiro, Nioh, Samurai sen) rather than when
they're the same verb at different stat lines.

## 3. Effect stacking, duration, and intensity

Tier 1/2/3 effect-proc design, poison-duration tricks, the kind of
thing in `BRAINDUMP.md`'s "Combat — Tier 2 Synergy" section.

| Game | Mechanic | Notes |
|---|---|---|
| Slay the Spire | Poison + Catalyst (doubles poison) | The "double the stack" play is iconic. Players love the *spike*. |
| Slay the Spire | Strength + Limit Break (doubles strength) | Same pattern as Catalyst. The doubling lever shows up enough to feel like a design language. |
| Path of Exile | Ailment magnitude (poison/ignite/bleed scale with hit damage) | Stack count + per-stack scaling = build-defining numbers. Loved by theorycrafters; "stack count is the only number" is a real player complaint. |
| MTG | Storm count | Each spell this turn pumps the next. One of the most-loved and most-banned mechanics simultaneously. |
| MTG | Cumulative upkeep | The opposite curve: stacks cost you more over time. Lukewarm reception — felt punishing. |
| Hades | Cast / Bloodstones with Aspect of Beowulf | Stack-up-then-spend pattern. Praised for satisfying release. |
| Risk of Rain 2 | Multiplicative item stacking | Loved for explosive late-game; criticized for trivializing balance. |

**Design takeaway:** Duration-based payoffs (like Tier 2 doubling poison
duration ≥ 5 in BRAINDUMP.md) read more like Slay the Spire's
Catalyst/Limit Break shape than like MTG storm count. Watch for "stack
count becomes the only number" — give other levers (intensity, type
conversion, side-effects).

## 4. Type / effect conversion

The "convert Thorns into Bat Swarm" pattern from `BRAINDUMP.md`.

| Game | Mechanic | Notes |
|---|---|---|
| MTG | Painter's Servant, Mycosynth Lattice | Changes the *color/type* of permanents. Loved for combo discovery; banned/restricted for breaking color identity. |
| Slay the Spire | Wraith Form, Echo Form, Doppelganger | Modify the *rules* of a card/turn rather than convert types. |
| Risk of Rain 2 | Items that "convert N stacks of X into Y" | Recipe-discovery feel. Praised. |
| Diablo III/IV | Skill rune systems | "Same skill, different element/effect". Praised when runes feel distinct, panned when they're +X variants. |
| Pokémon | Normalize ability, Soak move (changes type) | Niche but beloved by competitive players for breaking matchups. |

**Design takeaway:** Type-conversion mechanics shine when they enable
*combos the player invents*, not when designers spell them out. Risk:
if every effect can become every other effect, type identity collapses.

## 5. Morality-driven difficulty / branching

The BRAINDUMP.md "no upfront difficulty selection, meter driven by moral
choices" idea.

| Game | Mechanic | Notes |
|---|---|---|
| Undertale | Pacifist / Neutral / Genocide routes | Moral choices change combat *and* narrative *and* music *and* fourth-wall. The gold standard. Praised; "Genocide is harder" feels earned. |
| Pathologic 1 & 2 | Survival difficulty tied to moral/triage choices | Praised for thematic integrity; criticized as opaque/punishing. |
| Tyranny | Spires/Alignment dictates which spells you have | Morality literally gates mechanics. Loved by replay-focused players. |
| The Witcher 3 | Hidden morality consequence chains | Praised for "no obvious right answer"; some chains feel disconnected from inputs. |
| Disco Elysium | No combat difficulty per se; thought cabinet + skill voices shape outcomes | Failure is *interesting*. Aligns with Axiomancer's stated design philosophy. |
| Dishonored 1 & 2 | Chaos system tied to lethality | High Chaos = harder game *and* darker ending. Some players felt it punished playstyle. |
| NieR: Automata | Multiple endings; later runs unlock harder + truer content | Reception: legendary. |

**Design takeaway for Axiomancer:** the watch-out is "optimal morality
routing" — when players figure out the meter is just a difficulty
slider, they game it. Undertale survives this by making the
*narrative cost* of routing for difficulty enormous (Genocide
literally damages the save file's future). Bake the cost in.

## 6. Fallacy / paradox / philosophical flavor for skills

Axiomancer's signature: skills named after logical fallacies and
paradoxes.

| Game | Mechanic | Notes |
|---|---|---|
| Disco Elysium | 24 skills as voices in the protagonist's head (Inland Empire, Shivers, Logic, …) | Skills *talk to the player*; flavor IS the mechanic. Universally praised. |
| Pentiment | Skills tied to humanist disciplines (Theology, Logic, Medicine) | Choice gates dialog and investigation; praised for thematic integrity. |
| In Stars and Time | Skills tied to time-loop knowledge; emotional resonance gates power | Praised for theme-mechanics alignment. |
| Hand of Fate 1 & 2 | Cards as encounters; tarot-tinged framing | Mixed: theme is strong, mechanics are sometimes generic underneath. |
| Citizen Sleeper | Stat checks as dice-allocation framed by character class roles | Praised for tying mechanics to identity. |
| Inscryption | Sacrifice + sigils + meta-narrative wrapping | Praised for *theme as mechanic* — sigils evolve, are read as text. |

**Design takeaway:** Theme-as-mechanic survives when the skill's *name*
actually predicts what it does. "Strawman Argument" as a skill should
do something that maps to strawmanning (e.g. attack a false weaker
version of an enemy's threat). If the skill is just "Strawman
Argument: +5 damage", players bounce off — the theme becomes
decoration.

## 7. Quick / decisive combat

Mörk Borg-inspired short fights.

| Game | Mechanic | Notes |
|---|---|---|
| Mörk Borg | d20 attack vs. armor; HP intentionally low; "doom" track | Short violent fights respect player time. Beloved by OSR crowd. |
| Knave / Cairn | Stripped-down OSR-style HP & saves | Same praise pattern. |
| Into the Breach | Perfect-information tactical puzzles, 4-turn missions | Praised for surgical brevity. |
| Slay the Spire | Most fights ~3–6 turns | Praised; longer Act 3 fights are the ones community complains about. |
| Sekiro | Bosses are long, but mooks die in 1–2 hits | Praised; gives boss fights weight. |

**Design takeaway:** "decisive" is partly about *low HP totals* and
partly about *escalating costs* (Mörk Borg's doom, Slay the Spire's
deck dilution). One without the other tends to feel either swingy or
grindy.

## 8. Resource economy / commitment costs

Often comes up adjacent to stance-switching and effect design.

| Game | Mechanic | Notes |
|---|---|---|
| MTG | Mana curve + colors | The original "commit early, pay later" loop. |
| Slay the Spire | Energy + draw count | Tight per-turn budget = decisions matter. |
| Hades | Mana, cast bloodstones, death defiance | Multiple parallel small budgets. Praised for variety. |
| Disco Elysium | Skill points + thought cabinet slots | Slots are the scarcity; thoughts cost time. |
| Pathfinder 2e | 3-action economy per turn | Praised for clarity; criticized for solving every fight the same way. |

## How to use this catalog

When you brainstorm in Phase 3:

1. Identify which problem shape (1–8 above) the user's question fits.
2. Pull **2–4 references that solved it differently from each other**,
   not 4 variants of the same answer.
3. State the named mechanic, one sentence on what it does, one sentence
   on player reception. Skip the reception note if you don't know it.
4. Extend this file when a new problem shape comes up; future-you
   benefits.
