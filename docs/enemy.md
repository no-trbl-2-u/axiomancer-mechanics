# Enemies

> Adversaries you encounter in combat. Each enemy is a fully-stat'd combatant with an AI behaviour, a tier classification, and a map location. Enemy data lives in `src/Enemy/enemy.library.ts`; AI is in `src/Enemy/enemy.logic.ts`; encounters are generated by `src/Enemy/encounter.library.ts`.

## Enemy Type

```ts
interface Enemy {
    id: string;
    name: string;
    description: string;
    enemyTier?: 'simple' | 'normal' | 'elite' | 'boss' | 'unique';
    level: number;
    health: number;
    maxHealth: number;
    mana: number;
    maxMana: number;
    baseStats: BaseStats;
    derivedStats: DerivedStats;
    mapLocation: Pick<Map, 'name'>;
    logic: EnemyLogic;
    tier1Effects?: EnemyTier1EffectMap;
    skills?: Skill[];
    loot?: Item[];
    currentActiveEffects: ActiveEffect[];
}
```

Use `createEnemy(opts)` to construct one — derived stats and resource caps are computed automatically.

## Tiers

| Tier | Range | Notes |
|---|---|---|
| `simple` | Lv 1-2 | Trash mobs; usually `random` AI |
| `normal` | Lv 2-4 | Aggressive / defensive / balanced AI |
| `elite`  | Lv 4-6 | Strategic AI; often drops Tier 2 gear |
| `boss`   | Lv 6+ | Boss AI; multi-phase, drops Tier 3 gear |
| `unique` | One-of-a-kind story bosses |

## AI Strategies

Defined in `enemy.logic.ts`. Dispatcher: `dispatchEnemyLogic(enemy, ctx)`.

| Logic | Behaviour |
|---|---|
| `random` | Uniformly random stance + action |
| `aggressive` | Always attacks; favours strongest stat 70% of the time |
| `defensive` | Defends at high HP %, attacks at low HP % |
| `balanced` | Roughly 50/50 attack/defend with a stat-aware bias |
| `strategic` | Counters the player's last stance via Heart > Body > Mind > Heart |
| `boss` | Three-phase: balanced opener (≥75% HP) → strategic attacks (25-75%) → enraged (<25%) |

`EnemyLogicContext` carries the runtime info AI strategies need (`enemyHpPercent`, `playerLastStance`, `round`).

Lookups: `lookupEnemy(id)`, `getEnemiesByMap(mapName)`, `getEnemiesByTier(tier)`.

## Encounters

```ts
interface Encounter {
    id: string;
    mapName: string;
    recommendedLevel: number;
    enemies: Enemy[];
}

generateEncounter(mapName, playerLevel)   // closest-fit enemy within ±3 levels
listEncountersForMap(mapName)             // all enemies on a map
listAllEncounters()                       // global enumeration
```

`generateEncounter` picks the closest-level match for the player on a given map and returns `undefined` when no enemy is within ±3 levels.

## Tier 1 Effect Overrides

Each enemy can override the global Tier 1 stance effect map per stance + action via `tier1Effects`. This is how bosses customise their thematic openers (e.g. a Sophist enemy might apply a unique mind-mark).
