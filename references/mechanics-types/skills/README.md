# Skills Types

## PhilosophicalAspect
Alignment of a skill with one of the three core attributes (body, mind, heart).

## SkillType
Category of skill: fallacy (offensive), virtue (balanced), logic (defensive), rhetoric (persuasive), or meditation (supportive).

## FallacyType
Classification of fallacy-based skills into formal, informal, or cognitive bias categories.

## SkillLearningRequirement
Prerequisites for learning a skill including level and stat minimums.

## SkillCombatEffects
Contextual effects of using a skill in different combat scenarios (advantage, disadvantage, defended).

## Skill
Complete skill entity with costs, effects, and learning requirements.
