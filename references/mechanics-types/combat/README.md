# Combat Types

## CombatType
Three approaches to conflict resolution with cyclic advantage (Heart > Body > Mind > Heart).

## CombatActionType
Actions available in combat (attack or defend).

## AdvantageType
Result of type matchup determining dice roll advantage.

## CombatDecision
Player's choice for a combat round combining type and action.

## BattleLogEntry
Record of a single combat round including rolls, damage, and results.

## CombatState
Active combat session state tracking both combatants and battle history.

## CombatResolutionResult
Result of resolving a single combat round.
