# Python automation script for testing the Combat Simulator

This script is meant as an automation test to run the axiomancer-mechanics combat simulator a number of times to generate logs in order to debug certain issues found within.

> `npm run combat:auto x y z`
>
> -  "x": Number of times the `npm run combat` will run
> -  "y": "Stance" decision for every battle (default: _Random_)
> -  "z": "Action" decision for every battle (default: _Random_)
